## 코디 커뮤니티

사용자는 자신만의 스타일 코디 맵을 
만들어 커뮤니티에 공유할 수 있다

스타일 코디 맵에 사용되는 아이템은
1) 유저 옷장
2) 네이버 쇼핑
3) 네이버 쇼핑 유저 찜
